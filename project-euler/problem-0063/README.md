If n=3, then 100 ≤ x³ < 1000, which would give the values of X for n=3.  This corresponds to the formula 10^(n-1) ≤ x^n < 10^n.  Simplifying for x gives the upper and lower limits of x, 10^((n-1)/n) ≤ x < 10.
